# REQUISTI - Advertising Industry Directory

A modern directory platform for marketing and communication professionals, inspired by Scanbook/La Réclame.

## Getting Started

### Prerequisites

- Node.js 18+
- npm
--
### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

**Note for Windows PowerShell users:** If you need to change directory first, use `;` instead of `&&`:

```powershell
cd "C:\path\to\requisti-v0"; npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000) (or port 3001 if 3000 is in use).

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Project Structure

- `app/` - Next.js app router pages and layout
- `components/` - React components
- `lib/` - Mock data and utilities
